# CleverQuit - AI Quit Smoking Coach

This contains everything you need to run your CleverQuit app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1B2rd6YY3O2y9X_2t4rKdpYQD5etSd9bW

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
